Tentang Website

Website ini menerapkan model machine learning dengan fungsi mengklasifikasikan input berupa gambar kucing ke jenis ras kucing dengan probabilitas tertinggi. Akurasi model yang dipakai pada website ini ialah 87%. Website ini dibuat untuk menyelesaikan tugas akhir pembuat berupa penulisan ilmiah Jurusan Informatika Fakultas Teknologi Industri Universitas Guadarma.
